Turbo-Shell is a graphical-user-interface (GUI) for DOS that is useful for people who don't need, want, or can't afford Microsoft Windows or IBM OS/2. It comes with many useful utilities, too!

Key Benefits

  • Benefit 1: You can double-click onto a DOS program and run it. When you exit the program, the GUI (QD Shell) will re-appear.
  • Benefit 2: The QD Shell comes with many things that other popular operating systems come with.
  • Benefit 3: The GUI is useful for business and home users.

There is no version of the Turbo-Shell available yet, because it is a work-in-progress. When version 1.0 is compiled, it will be available for the public to get.
